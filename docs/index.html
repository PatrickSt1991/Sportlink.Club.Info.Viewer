Sportlink Club Info Viewer

A Vue.js application providing real-time match information.

Overview

The Sportlink Club Info Viewer is a Vue.js application that provides users with real-time information about upcoming matches over the next week, match results over the last week, and pre-match information including dressing room and field information. Built using Vue 3 and Vite, the app fetches match data from an external (Sportlink) API and displays it in a user-friendly interface.

Features

  • Real-time Data Fetching: Automatically retrieves match schedules for the next 7 days, past 7 days for match results, and the next 3 hours for pre-match information.
  • Responsive Design: Adapts to various screen sizes for optimal viewing on desktops, tablets, and mobile devices.
  • User Notifications: Displays loading messages and error alerts to keep users informed about the data retrieval status.
  • Scrolling Match Information: Automatically scrolls through match information for easy viewing.
  • Fallbacks: Provides default club logos in case specific team logos are unavailable.

Installation

  1. Clone the repository: 
    git clone https://github.com/PatrickSt1991/Sportlink.Club.Info.Viewer.git
    cd Sportlink.Club.Info.Viewer
  2. Install dependencies: 
    npm install
  3. Update config.js:
    • BASE_URL: Should be empty if you want to run it in the Tizen app.
    • CLIENT_ID: Should be the clientId of your club.
    • PROGRAMMA_DAGEN: Number of days ahead for upcoming matches.
    • UITSLAG_DAGEN: Number of days in the past for match results.
    • HOMESCREEN: Screen visible at http://localhost.
  4. Run the development server: 
    npm run dev
  5. Open the application: Navigate to http://localhost:5173 in your browser.

Usage

  • Upon launching, the main screen displays the upcoming match schedules for the next week.
  • Users can view match date, time, home and away team details, and competition type.
  • If there are no matches or an error occurs, appropriate messages will be displayed.

API Reference

The application fetches match data from the following endpoints:

  • MatchInfo - Sportlink API
  • MatchResults - Sportlink API
  • PreMatchInfo - Sportlink API
